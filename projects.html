Full Text Book Remix Project

The goal of this project is to make the full text of a book that is out of print available online. The text will be taken from a PDF using optical character recognition, reformatted using HTML and CSS, and styled for a modern viewing experience. Also included will be supplementary material, links, and a tribute to the author.

Website Remix Project

The goal of this project will be to remake, restyle a webpage that is no longer available on the web. Using archived information from the wayback machine, elements of the original site will be combined with updated styling and navigation using HTML and CSS, with additional links and tribute information to the author who's website is being resurrected.
